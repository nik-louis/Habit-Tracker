How to create a database (once, ~3 min) ▾
  1. Open console.firebase.google.com, sign in with Google.
  2. Create project → any name → disable analytics → create.
  3. Left panel: Build → Realtime Database → Create → any region → test mode.
  4. Rules tab → paste and publish:
{ "rules": { "sync": { "$c": { ".read": "$c.length >= 16", ".write": "$c.length >= 16" } } } }
  1. Copy the database URL (like https://name-default-rtdb...firebasedatabase.app) and paste below.

Данные шифруются на устройстве (AES-256). В облаке — нечитаемый шифротекст. Забудешь пароль — данные не восстановить.
